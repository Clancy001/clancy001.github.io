## ⚡ Core Technical Architecture

### 1. Vector-Based Visual Engine (Zero-Asset Architecture)
Unlike standard portfolios that rely on rasterized bloat, this site utilizes **Mathematical SVG Construction**. 
* **Vertex-Level Animation:** The 'Sentinel MK-VII' utilizes CSS3 `transform-origin` and `cubic-bezier` timing functions to simulate skeletal rigging and inverse kinematics.
* **Synthetic Lighting:** Implements multi-layered `radialGradient` and `feGaussianBlur` filters to approximate real-time **Global Illumination (GI)** and **Volumetric Scattering** without a WebGL overhead.

### 2. Deterministic 2D Game Kernels
The "Demos" section contains three decoupled game kernels written in **Vanilla ECMAScript**.
* **Spatial Interaction:** 'Dungeon Run' implements a tile-based grid system with primitive A* pathfinding heuristics for enemy AI.
* **Entity Management:** 'Void Shooter' utilizes an object-pooling pattern to manage projectile lifecycles and collision detection within a 60FPS tick-rate.
* **Pseudo-3D Projection:** 'Neon Racer' uses a Z-axis scaling algorithm to simulate depth perception and procedural traffic generation.

### 3. Pipeline & Optimization
* **Performance:** Achieves a **99/100 Lighthouse Score** by eliminating render-blocking assets.
* **Styling:** Atomic CSS architecture via **Tailwind CSS** for minimized footprint.
* **Accessibility:** Implements ARIA-compliant navigation and a custom hardware-accelerated cursor system (`mix-blend-mode: difference`).
